Product Management Case Study

EventConnect: An Event Networking App

A complete product teardown for a mobile application designed to solve networking inefficiencies at professional events through AI-powered matchmaking and instant contact exchange.

Project Deliverables

The Problem

Event attendees struggle to find relevant people, exchange contacts efficiently, and follow up effectively. Existing tools like LinkedIn are too slow for live events, leaving valuable networking opportunities to chance.

The Solution

EventConnect is a mobile app focused entirely on solving this problem. It uses AI-powered matchmaking to suggest relevant connections and an instant QR-code scan to exchange contacts, making networking simple and valuable.

My Role & Key Outcomes

  • Role: Product Manager (Conceptual Project)
  • Developed a complete Product Requirements Document (PRD) from scratch.
  • Created low-fidelity wireframes and user flow diagrams for the core user journey.
  • Defined a Go-To-Market strategy and key success metrics (KPIs).
  • Conducted a competitive analysis and used the RICE framework for feature prioritization.
